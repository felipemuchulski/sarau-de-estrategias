# Sarau de Estratégias

Projeto full stack desenvolvido para estruturar a solução digital do **Sarau de Estratégias**, com separação entre front-end e back-end no mesmo repositório.

## Estrutura do projeto

- `web/` → aplicação front-end
- `api/` → aplicação back-end

## Branches

- `main` → versão principal
- `hml` → ambiente de homologação
- `dev-web` → desenvolvimento front-end
- `dev-api` → desenvolvimento back-end

## Status

Projeto em fase inicial de estruturação de repositório e arquitetura.
